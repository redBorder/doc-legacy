  Nombres de archivos y/o rutas

  sistema de archivos, file para archivos y dir para

  Texto de cuadro de diálogo

  -Open Stack

  gestión y disponibilidad que ésta aporta.

   <title>Bienvenido al asistente para la implantación de la solución
  redBorder IPS </title>

  Una vez que haya descargado el archivo
  rb_InstallIPS_v.3_community, encontrará la imagen ISO que hace
  las veces de asistente para la implantación de la solución
  redBorder IPS.

  La solución para la prevención de intrusiones redBorder IPS
  consta de dos elementos:

  Manager: sistema central que recibe los eventos recogidos por
  cada uno de los sensores instalados.

  Sensores IPS: red de dispositivos que envían eventos al Manager
  según las reglas establecidas para cada uno de los sensores.

  La implantación de la solución redBorder IPS consta de tres
  pasos:

  Configuración del Manager

  Configuración de los sensores IPS

  Registro de los sensores en el Manager

  [-- Image: Image10 --]Advertencia

  Es MUY IMPORTANTE que primero se realice la configuración del
  Manager antes de configurar los sensores.

  El Manager debe estar operativo y en una red accesible a los
  sensores. Algunos de los procesos de configuración de los
  sensores dependen del acceso al Manager.

  Bienvenido al asistente para la implantación de la solución

  Madiante el comando rb_sysconf,usted accederá a un menú desde el
  cual realizar todas las configuraciones necesarias para poner en
  marcha la solución redBorder IPS en su versión Community.

  [-- Image: Image11 --]

  Opciones disponibles del menú del instalador
  --------------------------------------------

  Este es el menú principal del instalador de la solución
  redBorder IPS. En él se contemplan las opciones que le
  permitirán realizar todas las configuraciones, tanto del
  Manager como de los sensores para la puesta en funcionamiento
  de la solución redBorder IPS.

  Asimismo, encontrará otras opciones muy útiles relacionadas con
  el funcionamiento de su equipo.

  Opciones del menú principal del instalador redBorder IPS:

  Install Manager Community IPS: acceso al menú de configuración
  del Manager.

  Install Sensor Community IPS: acceso al menú de configuración y
  registro de los sensores IPS.

  Rescue installed System: modalidad de inicio de rescate para los
  casos en que existan problemas de inicio de su máquina. Por
  ejemplo, si se produce algún problema en un sensor y su máquina
  no arranca, se crea una partición temporal sistema paralelo
  (modo de rescate) que permita actuar sobre el sistema de origen
  y realizar cambios a bajo nivel mediante la línea de comandos.

  Boot from Local Drive: arranque desde disco local (por defecto).

  Memory Test: testeo de memoria disponible para la instalación
  del sistema.

  <title>Proceso de Configuración del Manager </title>

 <title>Funciones del Manager</title>
  ---------------------

  Denominamos Manager al sistema central que recibe los eventos
  recogidos por los sensores. Permite además, realizar las
  siguientes funciones:

  Configuración centralizada o individual de las reglas que rigen
  los sensores.

  Creación de gráficas a partir de los datos recogidos por los
  sensores.

  El proceso de configuración del Manager comprende las siguientes
  acciones:

  1) Descarga e instalación de los paquetes del redBorder Community
  Installer y ajustes básicos.

  2) Configuración del Manager (puesta en red):

  a) Configuración de la red de gestión: creación de un bond

  b) Configuración de la red de servicio: configuración DNS y dominio

  b) Configurar hora local
  ---------------------

c) Introducir la IP del Master Manager 
  --------------------------------------
